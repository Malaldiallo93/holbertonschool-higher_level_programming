The 1-my_list module
====================

Using MyList
------------------

First import MyList from the 1-my_list module:

    >>> MyList = __import__('1-my_list').MyList

Now use it:

    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> print(my_list)
    [1, 4, 2]
    >>> my_list.print_sorted()
    [1, 2, 4]

    >>> my_list.append(2.3)
    >>> my_list.append(1.3)
    >>> print(my_list)
    [1, 4, 2, 2.3, 1.3]
    >>> my_list.print_sorted()
    [1, 1.3, 2, 2.3, 4]

    >>> my_list.append(-1)
    >>> print(my_list)
    [1, 4, 2, 2.3, 1.3, -1]
    >>> my_list.print_sorted()
    [-1, 1, 1.3, 2, 2.3, 4]

    >>> my_list = MyList("string")
    >>> print(my_list)
    ['s', 't', 'r', 'i', 'n', 'g']
    >>> my_list.print_sorted()
    ['g', 'i', 'n', 'r', 's', 't']

    >>> my_list = MyList([3, 6, 2])
    >>> print(my_list)
    [3, 6, 2]
    >>> my_list.print_sorted()
    [2, 3, 6]

    >>> my_list = MyList(None)
